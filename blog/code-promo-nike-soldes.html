Questions Fréquentes (FAQ) — Nike Soldes

Les codes Nike soldes sont-ils applicables aux sneakers en édition limitée ?

Non, les codes comme SALE25 ou SNEAKER18 ne s'appliquent généralement pas aux produits en édition limitée ou aux préventes. Ces articles sont généralement vendus à prix fixe. Cependant, une fois le produit disponible dans le catalogue général (après la période de préventе), il peut devenir éligible aux soldes.

Comment obtenir le code MEMBER30 ?

Le code MEMBER30 est réservé aux membres Nike gratuits. Inscrivez-vous sur Nike.com ou via l'application Nike pour accéder automatiquement à ce code et à d'autres avantages membres.

Les retours sont-ils gratuits chez Nike ?

Oui, Nike offre les retours gratuits pour tous les achats. Vous disposez de 60 jours à compter de la date de livraison pour retourner les articles. Le processus de retour est simple : demandez un retour en ligne et imprimez l'étiquette de retour.

Peut-on utiliser plusieurs codes sur une même commande Nike ?

Non, Nike n'autorise qu'un seul code de réduction par commande. Si vous avez plusieurs codes, comparez leur application pour utiliser celui qui vous donne la meilleure réduction finale.

Les codes Nike fonctionnent-ils en boutique ?

Certains codes sont valables uniquement en ligne, d'autres en boutique, et certains fonctionnent partout. Vérifiez les conditions de chaque code sur PromosVault. En boutique, vous pouvez demander au vendeur d'appliquer un code en presenting le QR code depuis votre téléphone.
