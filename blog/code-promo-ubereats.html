Questions fréquentes (FAQ) — Uber Eats

Pourquoi mon code promos Uber Eats affiche "Non éligible" ?

Cela arrive généralement si le code est réservé aux nouveaux utilisateurs et que vous avez déjà passé commande, ou si le code est lié à un restaurant spécifique où vous ne commandez pas. Notre IA sur PromosVault s'efforce d'indiquer "Nouveau client" ou "Tout client" pour chaque code pour vous éviter cette erreur.

Et si ma commande Uber Eats arrive en retard ou froide ?

Uber Eats a mis en place en 2026 une garantie de délai. Si le livreur dépasse de plus de 15 minutes le délai estimé affiché lors du paiement, vous pouvez demander un dédommagement en crédits Uber Cash (souvent 5€) directement dans l'application. L'utilisation d'un code promos ne retire pas cette garantie.

L'abonnement Uber One est-il résiliable à tout moment ?

Oui, l'un des grands avantages de l'écosystème Uber est la souplesse. Vous pouvez tester Uber One gratuitement pendant un mois, profiter des frais de port offerts et des codes promos cumulés, puis résilier en un clic sans aucun frais caché.

Les codes promos fonctionnent-ils sur les boissons alcoolisées ?

En général, non. En raison de la législation française, les codes de réduction globaux ne s'appliquent pas sur les rayons alcool des supermarchés partenaires ou les cartes des bars. Cependant, ils s'appliquent sur tout le reste de votre repas.

Comment passer une commande groupée avec des amis ?

Utilisez la fonction "Commande Groupée" dans l'application. Elle permet à chacun de choisir son plat sur son propre téléphone. Attention : pour appliquer un code PromosVault, seul l'hôte (celui qui paie) peut saisir le code final. C'est un excellent moyen de partager les frais de service.

Que faire si mon code PromosVault ne s'applique pas au checkout ?

Vérifiez que vous ne l'avez pas déjà ajouté à votre compte au préalable. Si c'est le cas, Uber Eats l'appliquera automatiquement sans que vous n'ayez besoin de le retaper. Vérifiez également le montant minimum : certains codes demandent 15€ ou 20€ d'achat hors frais de service.
