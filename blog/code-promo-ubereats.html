Comment ajouter un code de réduction sur l'application Uber Eats ?

L'interface d'Uber Eats est très intuitive, mais il vaut mieux ajouter votre code sur votre compte avant de passer commande pour être certain de sa validité. Voici la méthode infaillible :

  1. Ouvrez l'application mobile Uber Eats.
  2. En bas à droite, appuyez sur "Compte".
  3. Faites défiler vers le bas et sélectionnez "Promotions".
  4. En haut à droite de l'écran "Promotions", appuyez sur "Ajouter une promotion".
