Comment utiliser un code promo sur Nike.com ?

  1. Sélectionnez vos articles sur nike.com/fr et ajoutez-les à votre panier
  2. Cliquez sur l'icône panier en haut à droite puis sur "Voir le panier"
  3. Dans la page panier, trouvez le champ "Code promo"
  4. Saisissez votre code PromoVault et cliquez sur "Appliquer"
  5. La réduction s'affiche immédiatement dans le récapitulatif
  6. Finalisez votre commande en cliquant sur "Passer à l'étape suivante"

Questions fréquentes sur les codes promos Nike

Les codes promos Nike fonctionnent-ils sur les sneakers en édition limitée ?

En règle générale, les codes promos Nike ne s'appliquent pas aux sneakers en édition limitée, aux collaborations exclusives (Jordan, Off-White, etc.) ni aux produits lancés depuis moins de 30 jours. Ces exclusions sont précisées dans les conditions de chaque code promo.

Peut-on utiliser un code promo Nike en boutique physique ?

Non, les codes promos PromoVault pour Nike sont exclusivement valables sur le site officiel nike.com/fr et sur l'application Nike. Ils ne sont pas acceptés dans les boutiques Nike physiques ni chez les revendeurs agréés.

Les codes promos Nike sont-ils valables sur les articles déjà en promotion ?

Cela dépend du code promo. Certains codes s'appliquent uniquement sur les articles au prix plein, tandis que d'autres peuvent se cumuler avec les promotions en cours. PromoVault indique clairement dans la description de chaque code s'il est cumulable avec d'autres offres.
