Questions fréquentes (FAQ) — Samsung

Est-ce que les codes promos Samsung fonctionnent sur tout le site ?

Le plus souvent, les codes comme SAMSUNG10 sont valables sur tout le catalogue à l'exception des produits en précommande (qui ont leurs propres codes) et des accessoires très spécifiques. Vérifiez toujours la description du code sur PromoVault pour connaître les exclusions éventuelles.

Comment fonctionne l'Offre de Remboursement (ODR) Samsung ?

C'est un remboursement différé. Vous payez votre produit sur le shop (avec un code promo), puis vous remplissez un dossier en ligne sur le portail Samsung Promotions avec votre preuve d'achat et le numéro de série de l'appareil. Le virement arrive sur votre compte bancaire sous 4 à 8 semaines.

Puis-je cumuler le code étudiant avec la reprise (Trade-in) ?

Oui, c'est l'un des rares cumuls autorisés. Le portail Samsung Campus applique une remise de base, et vous pouvez y ajouter un bonus de reprise pour votre ancien appareil. Cependant, les codes promos "publics" ne fonctionnent généralement pas sur le portail Campus déjà remisé.

Que se passe-t-il si j'annule une commande avec un code promo ?

Samsung vous remboursera le montant exact que vous avez payé après application de la remise. Si le code était à usage unique, il risque d'être perdu. Dans ce cas, contactez le service client Samsung ou revenez sur PromoVault pour trouver une alternative équivalente.

La livraison est-elle toujours gratuite chez Samsung ?

En 2026, Samsung propose la livraison standard gratuite sur tous les produits, sans aucun minimum d'achat. C'est un avantage majeur par rapport à certains concurrents. L'installation à domicile pour les gros appareils (TV, lave-linge) est également incluse gratuitement dans la plupart des cas.

Les points Samsung Rewards expirent-ils ?

Oui, les points Rewards ont une durée de validité limitée (généralement 12 à 24 mois). Pensez à les utiliser pour acheter des accessoires (chargeurs, coques) qui vous reviendront ainsi à 0€ en les cumulant avec un petit code promo.
