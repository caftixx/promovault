Foire Aux Questions (FAQ) sur Adidas

Un même code promo Adidas peut-il être utilisé deux fois ?

Sauf mention contraire (comme les événements Black Friday globaux), la majeure partie des coupons fournis par e-mail au compte (suite à une réclamation ou récompense de points) sont à usage unique. Cependant, certains petits codes, comme adidas5 vérifiés par l'IA PromoVault, opèrent de manière universelle sur de multiples paniers.

Pourquoi la remise ne s'applique pas sur ma paire de chaussures YEEZY ou Pharrell Williams ?

Chez Adidas, les collections limitées (Drops SNKRS, YZY, collaborations très prisées du type Bad Bunny ou Gucci) sont strictement régies par des conditions générales de vente excluantes : aucun code promo ne fonctionnera sur ce type d'articles, appelés "articles exclus de toute promotion" dans la description de chaque produit.

Peut-on associer plusieurs codes PromoVault sur une seule commande ?

Non, le système e-commerce d'Adidas empêche l'accumulation de plusieurs codes promotionnels dans la même zone de saisie.
