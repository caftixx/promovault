Questions fréquentes sur les codes promos Amazon

Peut-on cumuler plusieurs codes promos Amazon ?

En général, Amazon ne permet pas de cumuler plusieurs codes promos sur une même commande. Vous ne pouvez utiliser qu'un seul code promotionnel par achat. En revanche, certains codes sont cumulables avec les remises directes sur les produits ou les offres Prime.

Les codes promos Amazon fonctionnent-ils sur tous les produits ?

Pas nécessairement. Certains codes promos sont spécifiques à une catégorie de produits (mode, électronique, etc.) ou à un type de clients (membres Prime). Lisez toujours la description du code avant de l'utiliser pour vérifier les conditions d'éligibilité.

Un code promo Amazon peut-il être utilisé plusieurs fois ?

La plupart des codes promos Amazon sont à usage unique par compte client. Certains codes promotionnels peuvent être utilisés plusieurs fois, mais c'est indiqué dans les conditions d'utilisation. PromoVault précise toujours les conditions d'utilisation dans la description de chaque code.
