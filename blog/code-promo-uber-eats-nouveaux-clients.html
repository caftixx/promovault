Le code s'applique-t-il aux frais de livraison ?

Oui, Uber Eats propose régulièrement des codes pour les clients existants : offres de fidélité, promotions hebdomadaires, et codes spéciaux via l'application ou la newsletter. Surveillez les notifications de l'application pour ne pas les manquer.
