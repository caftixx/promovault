FAQ — Codes Promo Uber Eats Nouveaux Clients

Puis-je utiliser plusieurs codes sur la même commande ?

Non, Uber Eats ne permet qu'un seul code de réduction par commande en 2026. Cependant, vous pouvez cumuler un code promotionnel avec du crédit Uber Cash présent sur votre compte. La stratégie est donc d'utiliser NEWEATS15 sur la première commande et FOOD10 sur la seconde, puis votre crédit éventuel sur la troisième.

Le code WELCOME20 donne-t-il 20€ sur une seule ou deux commandes ?

Il est réparti sur deux commandes à hauteur de 10€ chacune. La première tranche de 10€ s'active dès la validation de votre première commande. La seconde tranche de 10€ apparaît dans votre section "Portefeuille" après livraison réussie de la première, et expire 7 jours après activation — ne tardez pas !

Que faire si mon code est refusé à la validation du panier ?

Vérifiez trois points : 1) La valeur de votre panier atteint bien le minimum requis, 2) le restaurant choisi est éligible (certains restaurants partenaires en mode "exclusif" peuvent exclure les codes tiers), 3) le code n'est pas expiré dans notre base. Si le problème persiste, contactez directement le support Uber Eats via le chat intégré dans l'application.

Les frais de livraison sont-ils inclus dans le calcul du minimum d'achat pour les codes ?

Non, les frais de livraison sont exclus du calcul. Le montant minimum de 20€ pour NEWEATS15 s'applique uniquement sur le prix des plats commandés. Les frais de livraison et les pourboires éventuels s'ajoutent en sus, après application de la réduction sur le sous-total alimentaire.

J'ai déjà un compte Uber (taxi). Puis-je quand même profiter des codes "nouveaux clients" Uber Eats ?

Techniquement non, car Uber et Uber Eats partagent le même système d'authentification. Cependant, si vous créez un compte avec une adresse email différente non liée à votre compte Uber, vous serez considéré comme un nouvel utilisateur Uber Eats. Cette pratique est dans une zone grise des CGU mais très couramment pratiquée.

Les codes fonctionnent-ils sur les commandes d'épicerie (Uber Eats Grocery) ?

La plupart des codes "Nouveaux Clients" listés ici s'appliquent sur l'ensemble des catégories disponibles sur la plateforme, y compris les épiceries, pharmacies et autres commerces. Vérifiez simplement dans les conditions du code la mention "Tous les partenaires" ou "Restaurants uniquement" avant de valider.
