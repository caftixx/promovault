Peut-on utiliser plusieurs fois le même code promos ?

Non, un code promos AliExpress (global) est généralement limité à une utilisation par compte client par événement. Quelle est la différence entre un code et un coupon vendeur ?

Le code (ex: IFPTSUXA) est fourni par AliExpress et fonctionne sur tout ou partie du catalogue. Le coupon vendeur est créé par le marchand lui-même. Non, les codes promos AliExpress s'appliquent uniquement sur la valeur des articles (le prix catalogue).
